# ScheduleApp

## アプリの概要
登録したurlを自動で起動できるアプリです.

## 利用例
* 時間割通りに講義のurlを登録しておき, クリックで素早くurlへ飛ぶ.
* abemaで放送するアニメのurlを登録し, 指定した時間に起動する.
* 1限目の講義のurlを登録し, 指定した時間に起動する.
